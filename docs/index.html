Claude Code te habla
en voz alta

Cuando Claude termina de trabajar, te lo dice con una voz casi humana: qué hizo, qué encontró y qué sigue. Tú mientras tanto, a lo tuyo.

Gratis · Cero tokens · Open source (MIT) · Aliada de la accesibilidad

Instálala en 1 minuto

git clone https://github.com/lowlufi/claude-voice.git
cd claude-voice && ./install.sh
voz test

Necesitas una Mac con Claude Code. Y listo: todas tus sesiones te hablarán solas. Se quita entero con ./uninstall.sh.

Todo el control, en una palabra

voz callateSe calla ya. También se calla sola al enviar un prompt o cuando vuelves a la terminal.
voz off / onApagar y encender, sin desinstalar.
voz motor neuralVoz moderna casi humana, gratis (con internet). voz motor tradicional vuelve a la de macOS, 100% offline.
voz demoEscucha las ~45 voces en español seguidas — cada una se presenta — y elige la tuya.
voz favoritas dalia victorGuarda tus favoritas; después basta voz usar dalia.
voz modo briefLeer solo lo esencial de cada respuesta.
voz velocidad 200Más rápido o más lento.

Lee las respuestas limpias (sin código ni tablas) · Avisa cuando Claude necesita permiso · Nunca habla encima de otra voz
