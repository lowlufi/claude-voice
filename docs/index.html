Claude Code te habla
en voz alta

Cuando Claude termina de trabajar, te lo dice con una voz casi humana: qué hizo, qué encontró y qué sigue. Tú mientras tanto, a lo tuyo.

Muestras reales del motor de voz neural.

Gratis · Cero tokens · Open source (MIT) · Aliada de la accesibilidad

Instálala en 1 minuto

git clone https://github.com/lowlufi/claude-voice.git
cd claude-voice && ./install.sh
voz test

Necesitas una Mac con Claude Code. Y listo: todas tus sesiones te hablarán solas. Se quita entero con ./uninstall.sh.

Tutorial: un comando a la vez

Todo se controla con la palabra voz desde cualquier terminal. Toca un comando para ver cómo se usa.

voz test — escuchar una prueba

Dice una frase con tu configuración actual, para comprobar que todo suena bien.

$ voz test
🔊 «Hola. Soy Claude, y a partir de ahora puedes escucharme.»
voz callate — que se calle ya

Corta lo que esté diciendo en ese instante. No apaga nada: la siguiente respuesta vuelve a sonar.

$ voz callate

También se calla sola al enviar un prompt y cuando vuelves a la terminal o al IDE.

voz off / voz on — apagar y encender

Silencio total sin desinstalar: los hooks quedan instalados pero mudos, hasta que la enciendas de nuevo.

$ voz off
Voz desactivada. Reactívala con: voz on
$ voz on
Voz activada.
voz motor — elegir voz moderna o tradicional

Dos motores: neural (casi humana, gratis, necesita internet) y tradicional (la de macOS, 100% offline).

$ voz motor neural       # instala lo necesario la primera vez
$ voz motor tradicional  # volver a la voz de macOS

Si estás sin internet, la neural cae sola a la tradicional — nunca te quedas sin voz.

voz voces — ver las voces disponibles

Lista las ~45 voces neuronales en español: México, Chile, España, Argentina y más.

$ voz voces
es-MX-DaliaNeural
es-CL-CatalinaNeural
es-PR-VictorNeural …
voz demo — escucharlas todas y elegir

Reproduce las voces una tras otra; cada una se presenta con su nombre y país. Ctrl+C para parar.

$ voz demo        # todas (~3 minutos)
$ voz demo es-MX  # solo las de un país
voz usar — cambiar de voz

Acepta voces neuronales, voces de macOS, o el nombre corto de una favorita.

$ voz usar es-CL-CatalinaNeural  # neural, nombre completo
$ voz usar dalia                 # favorita, nombre corto
$ voz usar Mónica                # voz de macOS (say -v '?')
voz favoritas — guardar tus preferidas

Guarda las que te gustaron de la demo; después basta el nombre corto para cambiar.

$ voz favoritas dalia victor
★ es-MX-DaliaNeural
★ es-PR-VictorNeural
$ voz usar victor
voz modo — cuánto quieres escuchar

full lee toda la respuesta (limpia, sin código ni tablas) · brief solo lo esencial y las preguntas · summary un resumen hecho con IA (único que gasta tokens) · off nada.

$ voz modo brief
voz velocidad — más rápido o más lento

En palabras por minuto. 185 es el punto de partida; 200–220 cuando te acostumbres.

$ voz velocidad 200
voz estado — ver cómo está todo

Muestra el motor, la voz, la velocidad, el modo y si está silenciada.

$ voz estado
motor:  edge (neural, con respaldo say)
voz neural: es-PR-VictorNeural …

Lee las respuestas limpias (sin código ni tablas) · Avisa cuando Claude necesita permiso · Nunca habla encima de otra voz
