Instalación en 3 pasos

Solo necesitas macOS y Claude Code. Sin APIs, sin claves, sin nada que pagar.

1

Clona el repositorio

git clone https://github.com/lowlufi/claude-voice.git

2

Ejecuta el instalador

./install.sh registra los hooks y crea el comando voz. Correrlo dos veces no rompe nada.

3

Escúchalo

voz test — y listo: cada nueva sesión de Claude Code te hablará sola.

Encender y apagar es un comando

La voz nunca estorba: se calla sola cuando escribes un prompt, y tú tienes el control total.

● Activada

voz on

Reactiva la lectura en voz alta. Es el estado por defecto después de instalar — no tienes que hacer nada.

● Silenciada

voz off

Apaga la voz por completo sin desinstalar nada. Los hooks siguen ahí, calladitos, hasta que digas voz on.

voz callate

Corta lo que esté diciendo ahora mismo. También se calla automáticamente cuando envías un nuevo prompt.

./uninstall.sh

¿Ya no la quieres? Un comando la quita entera: hooks, comando voz y archivos de estado.
