en voz alta

Cuando Claude termina de trabajar, te lo dice con voz: qué hizo, qué encontró y qué sigue. Tú mientras tanto, a lo tuyo.

Gratis · Sin internet · No gasta tokens · Open source (MIT)

¿Qué hace?

Tres cosas, y las hace solas — no tienes que tocar nada después de instalar.

🗣️

Te lee las respuestas

Al terminar cada tarea, lee el resultado en voz alta: sin código ni tablas, solo lo importante.

🔔

Te avisa si te necesita

«Claude necesita tu permiso para usar Bash» — te lo dice, para que no quedes esperando de más.

Se calla solo

En cuanto escribes un nuevo mensaje, la voz se detiene. Y con voz off queda en silencio.

Escúchala ahora

Esta demo usa la voz de tu navegador. En tu Mac sonará aún mejor, con las voces del sistema.

«Hola. Soy Claude, y a partir de ahora puedes escucharme.»

Instálala en 3 pasos

Toma menos de un minuto.

Necesitas: una Mac, Claude Code y python3 (si no lo tienes, ejecuta xcode-select --install).
1

Descarga el proyecto

git clone https://github.com/lowlufi/claude-voice.git
cd claude-voice
2

Ejecuta el instalador

Registra la voz en Claude Code y crea el comando voz. Es seguro correrlo más de una vez.

./install.sh
3

Pruébala

Deberías escuchar: «Hola. Soy Claude, y a partir de ahora puedes escucharme.»

voz test

Y listo: desde ahora, todas tus sesiones de Claude Code te hablarán solas.

Contrólala con una palabra

El comando voz funciona desde cualquier terminal. Estos son los que usarás:

voz callateSe calla ahora mismo.
voz off / voz onApagar y encender la voz (sin desinstalar nada).
voz motor neuralVoz moderna casi humana (gratis, necesita internet). Con voz motor tradicional vuelves a la de macOS, que funciona offline.
voz usar MónicaCambiar de voz — cualquier voz de tu Mac (say -v '?') o neural (voz voces lista ~40 en español, ej. es-CL-CatalinaNeural).
voz demoEscuchar todas las voces neuronales seguidas, cada una se presenta con su nombre y país — ideal para elegir la tuya (voz demo es-MX filtra por país).
voz velocidad 200Más rápido o más lento (palabras por minuto).
voz modo briefLeer solo lo esencial en vez de toda la respuesta.
voz estadoVer cómo está configurada.

💡 Consejo: las voces «Enhanced» de macOS suenan mucho más naturales y son gratis: Ajustes del Sistema → Accesibilidad → Contenido hablado → Gestionar voces.

♿ Pensada también para accesibilidad

claude-voice nació para no tener que mirar la terminal — y eso la vuelve útil para desarrolladores ciegos o con baja visión:

  • Las respuestas de Claude Code llegan en audio, automáticamente, sin atajos ni pasos extra.
  • Complementa a VoiceOver: el lector de pantalla para navegar, claude-voice para los resultados.
  • Voz, velocidad y cantidad de lectura configurables a lo que cada persona necesite.

¿Trabajas en accesibilidad y tienes ideas? Nos encantaría leerte.

Preguntas frecuentes

¿De verdad es gratis?
Sí. Usa el sintetizador de voz que ya viene en tu Mac: sin suscripciones, sin APIs, sin tokens, y funciona sin internet. (Solo el modo opcional «summary», apagado por defecto, usa unos pocos tokens para resumir respuestas largas.)
¿Funciona con otros asistentes además de Claude Code?
Está hecha para Claude Code. Pero el script acepta texto de cualquier programa: si tu herramienta puede ejecutar un comando, puede hablarte. Los ports son bienvenidos en GitHub.
¿En qué idiomas habla?
En el que quieras. Tiene dos motores: el de macOS (offline, por defecto en español con Paulina) y el neural (voz motor neural), gratis y con ~40 voces en español — de México, Chile, España, Argentina y más — además de decenas de otros idiomas.
¿Sirve en Windows o Linux?
Todavía no — depende del comando say de macOS. Es la contribución más pedida; si programas, anímate.
¿Cómo la desinstalo?
Con un comando: ./uninstall.sh. Quita todo sin dejar rastro y tu configuración de Claude Code queda intacta.
